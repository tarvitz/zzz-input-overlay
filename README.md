# Zenless Zone Zero Buttons Overlay

## Pre-requisites
You will need:

- [OBS](https://obsproject.com/)
- [Input Overlay Plugin](https://github.com/univrsal/input-overlay)

 Please follow these projects guides to setup your configuration.
